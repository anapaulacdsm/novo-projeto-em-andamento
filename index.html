Meu Kit de Ferramentas Digitais

HTML

A base de toda página da web. É uma linguagem de marcação usada para estruturar o conteúdo de sites, como; textos, imagens, links e vídeos. Com o HTML, organizo o que aparece na tela e dou sentido ao conteúdo para navegadores e motores de busca.

CSS

A arte da aparência na web. É a linguagem responsável por deixar os sites bonitos e organizados. Com o CSS, defino cores, fontes, tamanhos, espaçamentos e crio layouts que se adaptam a diferentes telas, como celulares e computadores.

JavaScript

O motor da web. É a linguagem que torna as páginas interativas e dinâmicas. Com o JavaScript, crio animações, botões que reagem ao clique, formulários inteligentes e muitas outras funcionalidades que deixam o site mais vivo e atraente para o usuário.
